# Next.js FakeStore Showcase

A simple Next.js (App Router) project that displays products from FakeStore API.

## Getting Started
1. Install dependencies
   ```bash
   npm install
   ```
2. Run development server
   ```bash
   npm run dev
   ```

Then open [http://localhost:3000](http://localhost:3000) to view it.

Deployed easily on Vercel!
